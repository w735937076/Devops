## 1. 任务目标

实现通知中心模块，支持企业微信、钉钉、邮件等多种通知渠道，提供事件驱动的通知发送能力，具备通知历史、送达追踪、频率控制和可扩展的事件类型。

### 4.1 事件分类

| 构建 | BUILD_START | 构建开始 | INFO |

| 部署 | DEPLOY_FAIL | 部署失败 | WARNING |
| 部署 | DEPLOY_APPROVAL | 待审批 | INFO |
| 回滚 | ROLLBACK_SUCCESS | 回滚成功 | WARNING |
| 回滚 | ROLLBACK_FAIL | 回滚失败 | CRITICAL |

| 安全 | LOGIN_FAILED | 登录失败 | WARNING |

### 6.2 企业微信渠道

```java
// com.drp.notify.channel.WeComChannel
@Component
@RequiredArgsConstructor
public class WeComChannel implements NotificationChannel {

    private final RestTemplate restTemplate;
    private static final int MAX_RETRY = 3;
    private static final Duration RETRY_DELAY = Duration.ofSeconds(30);

    @Override
    public NotificationChannelType getChannelType() {
        return NotificationChannelType.WECOM;
    }

    @Override
    public String getChannelName() {
        return "企业微信";
    }

    @Override
    public SendResult send(NotifyMessage message, JSONObject config) {
        String webhookUrl = config.getString("webhookUrl");

        // 构建消息内容 - 使用 Markdown 格式
        Map<String, Object> body = new HashMap<>();
        body.put("msgtype", "markdown");

        Map<String, Object> markdown = new HashMap<>();
        markdown.put("content", buildMarkdownContent(message));
        body.put("markdown", markdown);

        // 添加安全提示 (可选)
        if (config.containsKey("safe")) {
            body.put("safe", config.getInteger("safe"));
        }

        try {
            ResponseEntity<WeComResponse> response = restTemplate.postForEntity(
                webhookUrl, body, WeComResponse.class);

            if (response.getBody() != null && response.getBody().getErrCode() == 0) {
                return SendResult.success()
                    .setExternalId(response.getBody().getMsgId()); // 返回消息ID
            } else {
                String errMsg = response.getBody() != null
                    ? response.getBody().getErrMsg()
                    : "未知错误";
                return SendResult.fail("发送失败: " + errMsg);
            }
        } catch (Exception e) {
            return SendResult.fail("发送异常: " + e.getMessage());
        }
    }

    @Override
    public boolean test(JSONObject config) {
        try {
            NotifyMessage testMsg = NotifyMessage.builder()
                .title("测试通知")
                .content(buildTestContent())
                .severity(Severity.INFO)
                .build();
            SendResult result = send(testMsg, config);
            return result.isSuccess();
        } catch (Exception e) {
            log.error("企业微信测试失败", e);
            return false;
        }
    }

    @Override
    public JSONObject getConfigSchema() {
        JSONObject schema = new JSONObject();
        schema.put("webhookUrl", Map.of(
            "type", "string",
            "label", "Webhook地址",
            "required", true,
            "placeholder", "https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxx"
        ));
        schema.put("safe", Map.of(
            "type", "select",
            "label", "安全设置",
            "options", List.of(
                Map.of("label", "不加密", "value", 0),
                Map.of("label", "签名校验", "value", 1),
                Map.of("label", "IP地址", "value", 2),
                Map.of("label", "签名+IP", "value", 4)
            ),
            "default", 0
        ));
        return schema;
    }

    private String buildMarkdownContent(NotifyMessage message) {
        StringBuilder sb = new StringBuilder();
        sb.append("# ").append(getSeverityEmoji(message.getSeverity()))
          .append(" ").append(message.getTitle()).append("\n\n");
        sb.append(message.getContent()).append("\n\n");

        sb.append("---").append("\n\n");

        if (message.getProjectName() != null) {
            sb.append("- **项目**: ").append(message.getProjectName()).append("\n");
        }
        if (message.getEnv() != null) {
            sb.append("- **环境**: ").append(formatEnv(message.getEnv())).append("\n");
        }
        if (message.getVersion() != null) {
            sb.append("- **版本**: ").append(message.getVersion()).append("\n");
        }
        if (message.getOperator() != null) {
            sb.append("- **操作人**: ").append(message.getOperator()).append("\n");
        }
        sb.append("- **时间**: ").append(LocalDateTime.now().format(
            DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss"))).append("\n");

        return sb.toString();
    }

    private String getSeverityEmoji(Severity severity) {
        return switch (severity) {
            case INFO -> "ℹ️";
            case WARNING -> "⚠️";
            case CRITICAL -> "🔴";
        };
    }

    private String formatEnv(String env) {
        return switch (env) {
            case "dev" -> "🛠️ 开发环境";
            case "test" -> "🧪 测试环境";
            case "pre" -> "🚀 预发布环境";
            case "prod" -> "🏭 生产环境";
            default -> env;
        };
    }

    private String buildTestContent() {
        return "这是一条测试消息\n\n" +
               "如果您收到此消息，说明企业微信通知配置成功。\n\n" +
               "- **状态**: ✅ 正常\n" +
               "- **时间**: " + LocalDateTime.now().format(
                   DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss"));
    }
```

### 6.3 钉钉渠道 (完整实现)

```java
// com.drp.notify.channel.DingTalkChannel
@Component
@RequiredArgsConstructor
public class DingTalkChannel implements NotificationChannel {

    private final RestTemplate restTemplate;
    private final RedisTemplate<String, String> redisTemplate;

    private static final String DINGTALK_API = "https://oapi.dingtalk.com/robot/send";
    private static final String ACCESS_TOKEN_KEY = "dingtalk:accesstoken:";

    @Override
    public NotificationChannelType getChannelType() {
        return NotificationChannelType.DINGTALK;
    }

    @Override
    public String getChannelName() {
        return "钉钉";
    }

    @Override
    public SendResult send(NotifyMessage message, JSONObject config) {
        String webhookUrl = config.getString("webhookUrl");
        String secret = config.getString("secret");

        try {
            // 获取加签后的 URL
            String signUrl = buildSignedUrl(webhookUrl, secret);

            // 构建消息
            Map<String, Object> body = new HashMap<>();
            body.put("msgtype", "markdown");

            Map<String, Object> markdown = new HashMap<>();
            markdown.put("title", buildTitle(message));
            markdown.put("text", buildMarkdownContent(message));
            body.put("markdown", markdown);

            // 发送
            HttpHeaders headers = new HttpHeaders();
            headers.setContentType(MediaType.APPLICATION_JSON);

            ResponseEntity<DingTalkResponse> response = restTemplate.exchange(
                signUrl,
                HttpMethod.POST,
                new HttpEntity<>(body, headers),
                DingTalkResponse.class
            );

            if (response.getBody() != null && response.getBody().getErrCode() == 0) {
                return SendResult.success()
                    .setExternalId(String.valueOf(response.getBody().getTaskId()));
            } else {
                String errMsg = response.getBody() != null
                    ? response.getBody().getErrmsg()
                    : "未知错误";
                return SendResult.fail("发送失败: " + errMsg);
            }
        } catch (Exception e) {
            log.error("钉钉发送失败", e);
            return SendResult.fail("发送异常: " + e.getMessage());
        }
    }

    /**
     * 构建加签 URL
     * 签名算法: HMAC-SHA256 + Base64
     */
    private String buildSignedUrl(String webhookUrl, String secret) {
        if (secret == null || secret.isBlank()) {
            return webhookUrl;
        }

        try {
            long timestamp = System.currentTimeMillis();
            String stringToSign = timestamp + "\n" + secret;

            // 使用 Hutool 的 HmacUtil
            byte[] signBytes = cn.hutool.crypto.symmetric.SymmetricAlgorithm.HmacSm3
                .digest(stringToSign.getBytes(StandardCharsets.UTF_8));

            // 也可以使用 HMAC-SHA256
            // Mac mac = Mac.getInstance("HmacSHA256");
            // SecretKeySpec keySpec = new SecretKeySpec(secret.getBytes(StandardCharsets.UTF_8), "HmacSHA256");
            // mac.init(keySpec);
            // byte[] signBytes = mac.doFinal(stringToSign.getBytes(StandardCharsets.UTF_8));

            String sign = Base64.getEncoder().encodeToString(signBytes);
            sign = URLEncoder.encode(sign, StandardCharsets.UTF_8);

            String separator = webhookUrl.contains("?") ? "&" : "?";
            return webhookUrl + separator + "timestamp=" + timestamp + "&sign=" + sign;

        } catch (Exception e) {
            log.error("钉钉签名失败", e);
            return webhookUrl;
        }
    }

    @Override
    public boolean test(JSONObject config) {
        try {
            NotifyMessage testMsg = NotifyMessage.builder()
                .title("测试通知")
                .content(buildTestContent())
                .severity(Severity.INFO)
                .build();
            SendResult result = send(testMsg, config);
            return result.isSuccess();
        } catch (Exception e) {
            log.error("钉钉测试失败", e);
            return false;
        }
    }

    @Override
    public JSONObject getConfigSchema() {
        JSONObject schema = new JSONObject();
        schema.put("webhookUrl", Map.of(
            "type", "string",
            "label", "Webhook地址",
            "required", true,
            "placeholder", "https://oapi.dingtalk.com/robot/send?access_token=xxx"
        ));
        schema.put("secret", Map.of(
            "type", "string",
            "label", "加签密钥",
            "required", false,
            "placeholder", "安全设置中的加签密钥"
        ));
        return schema;
    }

    private String buildTitle(NotifyMessage message) {
        return getSeverityEmoji(message.getSeverity()) + " " + message.getTitle();
    }

    private String buildMarkdownContent(NotifyMessage message) {
        StringBuilder sb = new StringBuilder();
        sb.append("## ").append(message.getTitle()).append("\n\n");
        sb.append(message.getContent()).append("\n\n");
        sb.append("---\n\n");

        if (message.getProjectName() != null) {
            sb.append("**项目**: ").append(message.getProjectName()).append("\n\n");
        }
        if (message.getEnv() != null) {
            sb.append("**环境**: ").append(formatEnv(message.getEnv())).append("\n\n");
        }
        if (message.getVersion() != null) {
            sb.append("**版本**: ").append(message.getVersion()).append("\n\n");
        }
        if (message.getOperator() != null) {
            sb.append("**操作人**: ").append(message.getOperator()).append("\n\n");
        }
        sb.append("**时间**: ").append(LocalDateTime.now().format(
            DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss"))).append("\n\n");

        return sb.toString();
    }

    private String getSeverityEmoji(Severity severity) {
        return switch (severity) {
            case INFO -> "ℹ️";
            case WARNING -> "⚠️";
            case CRITICAL -> "🔴";
        };
    }

            case "test" -> "🧪 测试环境";
            case "pre" -> "🚀 预发布环境";
```

## 7. 频率控制服务

```java
// com.drp.notify.service.RateLimitService
@Service
@RequiredArgsConstructor
public class RateLimitService {

    private final RateControlRepository rateControlRepository;

    /**
     * 检查是否应该发送通知
     * @return true 如果可以发送，false 如果应该抑制
     */
    public boolean shouldSend(NotifyEvent event, String channel, String recipient) {
        // 查询频率控制记录
        Optional<RateControl> controlOpt = rateControlRepository.findByEventAndChannelAndRecipient(
            event.getEvent(), channel, recipient, event.getProjectId());

        if (controlOpt.isEmpty()) {
            return true; // 没有记录，可以发送
        }

        RateControl control = controlOpt.get();
        long windowSeconds = Duration.between(control.getWindowStart(), LocalDateTime.now()).getSeconds();

        // 检查是否在同一时间窗口内
        if (windowSeconds < 60) {
            // 1分钟内，同一事件+渠道+接收人，只发一次
            if (control.getSendCount() >= 1) {
                return false;
            }
        } else if (windowSeconds < 3600) {
            // 1小时内，最多发3次
            if (control.getSendCount() >= 3) {
                return false;
            }
        }

        return true;
    }

    /**
     * 记录发送
     */
    @Transactional
    public void recordSend(NotifyEvent event, String channel, String recipient) {
        RateControl control = rateControlRepository.findByEventAndChannelAndRecipient(
            event.getEvent(), channel, recipient, event.getProjectId())
            .orElseGet(() -> {
                RateControl newControl = new RateControl();
                newControl.setEvent(event.getEvent());
                newControl.setChannel(channel);
                newControl.setRecipient(recipient);
                newControl.setProjectId(event.getProjectId());
                newControl.setWindowStart(LocalDateTime.now());
                return newControl;
            });

        control.setLastSendTime(LocalDateTime.now());
        control.setSendCount(control.getSendCount() + 1);

        // 每小时重置计数
        if (Duration.between(control.getWindowStart(), LocalDateTime.now()).getSeconds() > 3600) {
            control.setWindowStart(LocalDateTime.now());
            control.setSendCount(1);
        }

        rateControlRepository.save(control);
    }
}
```

## 8. 重试服务

```java
// com.drp.notify.service.RetryService
@Service
@RequiredArgsConstructor
public class RetryService {

    private final NotificationRecordRepository recordRepository;
    private final NotificationService notificationService;
    private final NotificationChannelFactory channelFactory;

    private static final int MAX_RETRY = 3;
    private static final Duration[] RETRY_DELAYS = {
        Duration.ofSeconds(30),
        Duration.ofMinutes(1),
        Duration.ofMinutes(5)
    };

    /**
     * 异步重试失败的通知
     */
    @Async("notifyRetryExecutor")
    public void retryFailed(Long recordId) {
        NotificationRecord record = recordRepository.findById(recordId).orElse(null);
        if (record == null || record.getRetryCount() >= MAX_RETRY) {
            return;
        }

        try {
            // 获取配置
            NotificationConfig config = getConfig(record.getConfigId());
            if (config == null || !config.getEnabled()) {
                return;
            }

            // 重新构建消息
            NotifyMessage message = rebuildMessage(record);

            // 获取渠道
            NotificationChannel channel = channelFactory.getChannel(record.getChannel());

            // 重试发送
            SendResult result = channel.send(message, JSON.parseObject(config.getConfig()));

            // 更新记录
            record.setRetryCount(record.getRetryCount() + 1);
            if (result.isSuccess()) {
                record.setStatus(NotificationStatus.SUCCESS.name());
                record.setSentTime(LocalDateTime.now());
                record.setExternalId(result.getExternalId());
            } else {
                record.setErrorMessage(result.getErrorMessage());
                if (record.getRetryCount() >= MAX_RETRY) {
                    record.setStatus(NotificationStatus.FAIL.name());
                } else {
                    // 安排下一次重试
                    scheduleRetry(recordId, record.getRetryCount());
                }
            }
            recordRepository.save(record);

        } catch (Exception e) {
            log.error("重试失败", e);
            record.setErrorMessage(e.getMessage());
            record.setStatus(NotificationStatus.FAIL.name());
            recordRepository.save(record);
        }
    }

    /**
     * 安排重试
     */
    @Async("notifyRetryExecutor")
    public void scheduleRetry(Long recordId, int currentRetry) {
        if (currentRetry >= MAX_RETRY) {
            return;
        }

        Duration delay = RETRY_DELAYS[Math.min(currentRetry, RETRY_DELAYS.length - 1)];

        CompletableFuture.delayedExecutor(delay.toSeconds(), TimeUnit.SECONDS)
            .execute(() -> retryFailed(recordId));
    }

    private NotifyMessage rebuildMessage(NotificationRecord record) {
        return NotifyMessage.builder()
            .title(record.getTitle())
            .content(record.getContent())
            .recipient(record.getRecipient())
            .build();
    }

    @Data
    public static class SendResult {
        private boolean success;
        private String errorMessage;
        private String externalId;

        public static SendResult success() {
            SendResult result = new SendResult();
            result.setSuccess(true);
            return result;
        }

        public static SendResult fail(String message) {
            SendResult result = new SendResult();
            result.setSuccess(false);
            result.setErrorMessage(message);
            return result;
        }
    }
}
```

```java
// com.drp.notify.service.NotificationServiceImpl
@Service
@RequiredArgsConstructor
public class NotificationServiceImpl implements NotificationService {

    private final NotificationConfigRepository configRepository;
    private final NotificationRecordRepository recordRepository;
    private final NotificationChannelFactory channelFactory;
    private final RateLimitService rateLimitService;
    private final RetryService retryService;
    private final ObjectMapper objectMapper;

    @Override
    @Async("notifyExecutor")
    public void sendNotification(NotifyEvent event) {
        // 1. 查找匹配的配置
        List<NotificationConfig> configs = findConfigs(event);

        for (NotificationConfig config : configs) {
            if (!config.getEnabled()) continue;

            // 2. 频率控制检查
            if (!rateLimitService.shouldSend(event, config.getChannel(), config.getRecipients())) {
                log.info("通知被频率控制抑制 | event: {} | channel: {} | recipient: {}",
                    event.getEvent(), config.getChannel(), config.getRecipients());
                continue;
            }

            // 3. 获取渠道
            NotificationChannel channel = channelFactory.getChannel(config.getChannel());
            if (channel == null) {
                log.warn("未找到通知渠道: {}", config.getChannel());
                continue;
            }

            // 4. 构建消息
            NotifyMessage message = buildMessage(event, config);

            // 5. 发送并记录
            JSONObject channelConfig = JSON.parseObject(config.getConfig());
            SendResult result = channel.send(message, channelConfig);

            // 6. 保存发送记录
            saveRecord(config, message, result);

            // 7. 记录频率控制
            rateLimitService.recordSend(event, config.getChannel(), config.getRecipients());

            // 8. 如果失败，触发重试
            if (!result.isSuccess()) {
                NotificationRecord record = recordRepository
                    .findLatestByConfigIdAndRecipient(config.getId(), message.getRecipient())
                    .orElse(null);
                if (record != null) {
                    retryService.scheduleRetry(record.getId(), 0);
                }
            }
        }
    }

    private List<NotificationConfig> findConfigs(NotifyEvent event) {
        List<NotificationConfig> configs = new ArrayList<>();

        // 项目级配置优先
        if (event.getProjectId() != null) {
            configs.addAll(configRepository.findByProjectIdAndEventAndEnabled(
                event.getProjectId(), event.getEvent(), true));
        }

        // 全局配置
        configs.addAll(configRepository.findByProjectIdIsNullAndEventAndEnabled(
            event.getEvent(), true));

        return configs;
    }

    private NotifyMessage buildMessage(NotifyEvent event, NotificationConfig config) {
        String title = event.getTitle();
        String content = event.getContent();

        // 如果有模板，使用模板渲染
        if (config.getTemplateId() != null) {
            Template template = getTemplate(config.getTemplateId());
            if (template != null) {
                title = renderTemplate(template.getTitleTemplate(), event);
                content = renderTemplate(template.getContentTemplate(), event);
            }
        }

        return NotifyMessage.builder()
            .title(title)
            .content(content)
            .projectName(event.getProjectName())
            .env(event.getEnv())
            .version(event.getVersion())
            .operator(event.getOperator())
            .severity(event.getSeverity())
            .recipient(config.getRecipients())
            .build();
    }

    private String renderTemplate(String templateStr, NotifyEvent event) {
        // 简单的模板变量替换
        String result = templateStr;
        result = result.replace("{{projectName}}", nvl(event.getProjectName()));
        result = result.replace("{{env}}", nvl(event.getEnv()));
        result = result.replace("{{version}}", nvl(event.getVersion()));
        result = result.replace("{{operator}}", nvl(event.getOperator()));
        result = result.replace("{{content}}", nvl(event.getContent()));
        result = result.replace("{{time}}", LocalDateTime.now().format(
            DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss")));
        return result;
    }

    private String nvl(String str) {
        return str != null ? str : "-";
    }

    private void saveRecord(NotificationConfig config, NotifyMessage message, SendResult result) {
        NotificationRecord record = new NotificationRecord();
        record.setConfigId(config.getId());
        record.setChannel(config.getChannel());
        record.setEvent(config.getEvent());
        record.setRecipient(message.getRecipient());
        record.setTitle(message.getTitle());
        record.setContent(message.getContent());
        record.setStatus(result.isSuccess()
            ? NotificationStatus.SUCCESS.name()
            : NotificationStatus.FAIL.name());
        record.setErrorMessage(result.getErrorMessage());
        record.setExternalId(result.getExternalId());
        record.setSentTime(LocalDateTime.now());

        recordRepository.save(record);
    }
}
```

## 14. 风险与依赖

| 风险/依赖 | 影响 | 应对措施 |
|----------|------|---------|
| Webhook不可达 | 通知失败 | 自动重试 + 记录失败原因 |
| 限流 | 发送被限制 | 频率控制 + 队列缓冲 |
| 钉钉签名算法 | 签名失败 | 使用标准 HMAC-SHA256 |
| 邮件发送延迟 | 通知不及时 | 异步发送 + 状态追踪 |
