## 1. 任务目标

实现项目管理模块，提供项目创建、Git 配置（凭证中心）、环境变量管理、分支策略配置等功能。

**设计亮点：**
- 凭证管理抽离：Git Token、SSH 私钥等敏感信息统一由 `sys_credentials` 表管理
- 构建配置JSON化：不同语言（JAVA/NODE/PYTHON）的差异化参数通过 `build_config` JSON 字段存储
- 分支策略灵活化：通过正则表达式配置分支匹配规则，支持自动部署和审批流程

## 3. 数据库表设计

### 3.1 凭证管理表 (sys_credentials)

统一管理 Git Token、SSH 私钥等敏感信息，项目通过 `credential_id` 关联。

```sql
CREATE TABLE `sys_credentials` (
    `id` BIGINT NOT NULL AUTO_INCREMENT COMMENT '凭证ID',
    `name` VARCHAR(100) NOT NULL COMMENT '凭证名称(如：公司主Git账号)',
    `type` VARCHAR(20) NOT NULL COMMENT '类型：USERNAME_PASSWORD, ACCESS_TOKEN, SSH_KEY',
    `account` VARCHAR(100) DEFAULT NULL COMMENT '账号/标识',
    `secret_content` TEXT NOT NULL COMMENT '加密存储的密码、Token或私钥',
    `description` VARCHAR(255) DEFAULT NULL COMMENT '描述',
    `create_time` DATETIME DEFAULT CURRENT_TIMESTAMP,
    `update_time` DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    PRIMARY KEY (`id`)
) ENGINE=InnoDB COMMENT='凭证管理表';
```

### 3.2 项目基础信息表 (prj_project)

```sql
CREATE TABLE `prj_project` (
    `id` BIGINT NOT NULL AUTO_INCREMENT COMMENT '项目ID',
    `name` VARCHAR(100) NOT NULL COMMENT '项目名称',
    `code` VARCHAR(50) NOT NULL COMMENT '项目编码(用于Linux目录名，全局唯一)',
    `type` VARCHAR(20) NOT NULL COMMENT '类型：JAVA_MAVEN, NODE, PYTHON',
    `description` VARCHAR(500) DEFAULT NULL COMMENT '项目描述',

    -- Git 配置：关联凭证中心
    `git_url` VARCHAR(500) NOT NULL COMMENT 'Git仓库地址',
    `credential_id` BIGINT DEFAULT NULL COMMENT '关联凭证ID',
    `default_branch` VARCHAR(100) DEFAULT 'master' COMMENT '默认构建分支',

    -- 构建配置：JSON存储差异化参数
    `build_config` JSON DEFAULT NULL COMMENT '构建参数配置',

    -- 状态与审计
    `status` TINYINT DEFAULT 1 COMMENT '状态：0-禁用, 1-启用',
    `version` INT DEFAULT 1 COMMENT '乐观锁版本号',
    `is_deleted` TINYINT DEFAULT 0 COMMENT '逻辑删除',
    `create_user` BIGINT DEFAULT NULL COMMENT '创建者ID',
    `create_time` DATETIME DEFAULT CURRENT_TIMESTAMP,
    `update_time` DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    PRIMARY KEY (`id`),
    UNIQUE KEY `uk_code` (`code`)
) ENGINE=InnoDB COMMENT='项目基础信息表';
```

**设计说明：**
- `credential_id` 关联 `sys_credentials`，替代原有的 `git_credentials` 字段
- `build_config` JSON 字段存储差异化参数：
  - JAVA: `{"jdk": "17", "maven": "3.8", "goals": "clean package"}`
  - NODE: `{"node": "18.x", "npm": "9.x", "scripts": "build"}`

```java

    private String name;                    // 凭证名称

    private String type;                    // USERNAME_PASSWORD, ACCESS_TOKEN, SSH_KEY

    private String account;                 // 账号/标识（如 GitHub 用户名）

    private String secretContent;           // 加密后的密码/Token/私钥
```

### 6.3 凭证服务

```java
@Service
@RequiredArgsConstructor
public class CredentialService {

    private final CredentialRepository credentialRepository;
    private final AesEncryptUtil encryptUtil;  // AES 加密工具

    /**
     * 创建凭证（加密存储）
     */
    @Transactional
    public CredentialDTO create(CredentialCreateRequest request) {
        Credential credential = new Credential();
        credential.setName(request.getName());
        credential.setType(request.getType());
        credential.setAccount(request.getAccount());
        credential.setDescription(request.getDescription());

        // 加密敏感内容
        credential.setSecretContent(encryptUtil.encrypt(request.getSecretContent()));
        credential.setCreateTime(LocalDateTime.now());

        credentialRepository.insert(credential);
        return toDTO(credential, false);
    }

    /**
     * 获取凭证（脱敏返回）
     */
    public CredentialDTO getById(Long id) {
        Credential credential = credentialRepository.selectById(id);
        if (credential == null) {
            throw new BusinessException("凭证不存在");
        }
        // 不返回 secretContent
        return toDTO(credential, false);
    }

    /**
     * 获取解密后的凭证（供构建时使用）
     */
    public String getDecryptedContent(Long id) {
        Credential credential = credentialRepository.selectById(id);
        if (credential == null) {
            throw new BusinessException("凭证不存在");
        }
        return encryptUtil.decrypt(credential.getSecretContent());
    }

    private CredentialDTO toDTO(Credential credential, boolean includeSecret) {
        CredentialDTO dto = new CredentialDTO();
        dto.setId(credential.getId());
        dto.setName(credential.getName());
        dto.setType(credential.getType());
        dto.setAccount(credential.getAccount());
        dto.setDescription(credential.getDescription());
        dto.setCreateTime(credential.getCreateTime());
        if (includeSecret) {
            dto.setSecretContent(credential.getSecretContent());
        }
        return dto;
    }
}
```

```java
        return Result.success(credentialService.create(request));

        return Result.success(credentialService.update(id, request));

        credentialService.delete(id);
        return Result.success();
    }
}
```

```java
public boolean matches(String branchName, String pattern) {
    // 将通配符转换为正则
    String regex = pattern
        .replace(".", "\\.")
        .replace("*", ".*");
    return branchName.matches(regex);
}

// 使用示例
policy.matches("release/v1.0", "release/*")  // true
policy.matches("hotfix/bug", "release/*")    // false
```
